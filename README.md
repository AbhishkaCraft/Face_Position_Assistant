# Face Position Assistant

A real-time face positioning assistant built using Python and OpenCV.

## Features

- Detects faces using a webcam
- Draws a bounding box around the face
- Finds the face center
- Guides the user with:
  - Move Left
  - Move Right
  - Move Up
  - Move Down
  - Come Closer
  - Move Back
  - Perfect Position

## Technologies Used

- Python
- OpenCV
- NumPy

## How to Run

1. Install the required libraries:

```bash
pip install opencv-python numpy
```

2. Run the project:

```bash
python face_detection.py
```
